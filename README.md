# MCLauncherSharpLib

A library for managing Minecraft profiles and settings.

author : sorairo-rlf

Latest Version : ver 1.1.0

## Install

It can be installed using the NuGet package manager.

```bash
dotnet add package MCLauncherSharpLib
```
## Prerequisites
- .NET 8.0
- Windows environment recommended
